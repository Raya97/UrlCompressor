LIONESS API v2 - URL Shortener & Notes

Version: 2.0.0

Сервіс дозволяє реєструватися, логінитися, створювати короткі посилання, керувати нотатками, авторизуватися через JWT та використовувати різні ролі (USER, MODERATOR, ADMIN).

Endpoints:

/auth/register

POST: Реєстрація нового користувача

  • Tags: auth
  • Responses:
    • 201: Користувач успішно зареєстрований
    • 409:

/auth/login

POST: Авторизація користувача

  • Tags: auth
  • Responses:
    • 200: Успішна авторизація з JWT
    • 401:

/auth/refresh

POST: Оновлення JWT токену

  • Tags: auth
  • Responses:
    • 200: Токен успішно оновлено
    • 401:

/auth/logout

POST: Логаут користувача

  • Tags: auth
  • Responses:
    • 200: Вихід успішний, токен додано до чорного списку

/url

POST: Створення короткого URL

  • Tags: url
  • Responses:
    • 201:
    • 400:

/url/{shortUrl}

GET: Отримати повний URL за скороченим

  • Tags: url
  • Responses:
    • 200: Оригінальний URL отримано
    • 404:

/stats/{shortUrl}

GET: Отримати статистику за скороченим URL

  • Tags: url
  • Responses:
    • 200: Статистика отримана
    • 404:

/notes

GET: Отримати нотатки

  • Tags: notes
  • Responses:
    • 200: Усі нотатки користувача отримано

POST: Створити нотатку

  • Tags: notes
  • Responses:
    • 201: Нотатку створено

PUT: Оновити нотатку

  • Tags: notes
  • Responses:
    • 200: Нотатку оновлено

DELETE: Видалити нотатку

  • Tags: notes
  • Responses:
    • 200: Нотатку видалено

/admin/test

GET: Перевірка ролі ADMIN

  • Tags: admin
  • Responses:
    • 200: Доступ ADMIN підтверджено

/moderator/test

GET: Перевірка ролі MODERATOR

  • Tags: moderator
  • Responses:
    • 200: Доступ MODERATOR підтверджено
